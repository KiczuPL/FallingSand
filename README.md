# FallingSand

Simple falling sand game made
as individual project for WUT.
Built with Java and LibGDX framework.

Controls:
* Pause -> SPACE
* Speed up -> '>'
* Speed down -> '<'
* Spawn cells -> left click
* Open cell menu -> right click
* Shrink brush -> scroll down
* Enlarge brush -> scroll up
